# WhatsApp Template Example

Use this as a starting point in Meta WhatsApp Manager.

Template name:

```text
wedding_invitation
```

Language:

```text
Arabic
```

Category:

```text
Marketing
```

Header:

```text
Image
```

Body:

```text
السلام عليكم ورحمة الله وبركاته
يشرفنا دعوتكم لحضور حفل الزواج.
المدعو الكريم: {{1}}
حضوركم يسعدنا ويشرفنا.
```

Variable example for `{{1}}`:

```text
الأخ / أحمد سالم عبدالله
```

Important: If Meta rejects the template, simplify the wording and keep it clearly informational/social, without sales language.
